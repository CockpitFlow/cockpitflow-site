QUICK START

Setup in 2 minutes

1. Install Desktop

Download and run the installer. Launch CockpitFlow. It will connect to X-Plane or MSFS automatically.

2. Install Mobile

Download the APK and install on your Android device. Open the app — it will find your desktop automatically.

3. Fly

Select Cockpit or Checklist on mobile. Controls and data sync live with your simulator.
