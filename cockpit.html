CONTROLS

Everything at your fingertips

Yoke — Touch + Gyro

Touch pad with spring-back or tilt your device like a real yoke. Sensitivity and deadzone adjustable. Orientation-aware.

🔼

Levers

Vertical sliders for throttle, mixture, carb heat. Color-coded thumbs. Drag with your finger — smooth and responsive.

📈

Live Gauges

IAS, altitude, VS, heading, RPM — live from the sim. Color-coded warnings (green/amber/red).

🔘

Switches

Toggle switches for master, lights, fuel pump, avionics. Pill-style with red (master) and blue (systems) variants.

🔧

Flaps & Trim

Flap detent buttons (UP/10/20/FULL). Trim up/down with auto-repeat. All mapped to sim commands.

🔑

Magneto

OFF/R/L/BOTH/START positions. Spring-back on START. Works like the real thing.

LAYOUT ENGINE

Grid-based, fully customizable

Every cockpit layout is a JSON file with a grid (12×8 default). Each component has a position (col, row) and size (w, h). Design in the desktop app, fly on mobile.

{
  "grid": { "cols": 12, "rows": 8 },
  "components": [
    { "type": "lever", "col": 1, "row": 1, "w": 1, "h": 7, "cmd": "THROTTLE" },
    { "type": "yoke", "col": 5, "row": 1, "w": 5, "h": 7, "modes": ["touch", "gyro"] },
    { "type": "gauge", "col": 10, "row": 1, "w": 3, "h": 1, "dataref": "AIRSPEED" }
  ]
}
