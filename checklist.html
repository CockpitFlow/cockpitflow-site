FEATURES

More than a paper checklist

🔄

Two Modes

Strict — items in order, current highlighted, rest locked. Smart — check any item in any order.

⚡

Auto-Detect

Items auto-check when the sim detects the correct state. Supports both X-Plane datarefs and MSFS SimVars.

🟢

Feedback Dots

Green/red dots show real-time sim state for each item. See at a glance what's done and what needs attention.

📱

Mobile Companion

Checklist runs on your phone or tablet. Desktop controls settings, mobile just shows the checklist — clean and touch-optimized.

✈

Any Aircraft

Community presets for Cessna 172, Boeing 747, Airbus A320, and more. Or create your own with the visual editor.

💾

JSON-Based

Everything is a JSON file. Edit in any text editor, share on Discord, submit to GitHub. No proprietary format.

HOW IT WORKS

Desktop configures, mobile displays

Settings live in the desktop app. The mobile companion just shows the active checklist — no menus, no distractions.

On Desktop

Choose aircraft preset. Set mode (strict/smart). Enable auto-detect and feedback dots. Everything syncs to mobile automatically.

On Mobile

Open CockpitFlow app. Tap Checklist. Phase tabs at top, items below. Tap to check. How-to guide on tap. Progress at bottom.

DUAL SIMULATOR SUPPORT

X-Plane and MSFS in one file

Each auto-detect item can have both an X-Plane dataref and an MSFS SimVar. The app uses the right one based on which sim is connected.

{
  "label": "Master Switch",
  "expected": "ON",
  "auto_detect": {
    "condition": "> 0",
    "xplane_dataref": "sim/cockpit/electrical/battery_on",
    "msfs_simvar": "ELECTRICAL MASTER BATTERY"
  }
}
